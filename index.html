Planner App 1.0

(a fun project)

Semi-transparent todo & notes app for windows made with Electron. It always stays on top, which makes taking notes much easier without the need to switch back between the apps. The semi-transparent background and small UI makes the app a lot more convenient to use as it won't get in the way and block the view behind it. It also has 7 different backgrounds.

First page "To Do" is simply a Todo task list, where user can add or delete tasks. When clicking on the task, it moves into another page, "Done":

Second page "Done" is a finished tasks list. There can be found crossed out tasks from "To Do" page:

Last page "Notes" lets the user take quick notes. Due to the app being always on top it helps the user with visibility the other apps while taking notes:

Portfolio 1

(current)

This professional portfolio was inspired by windows 98. It is an interactive website to look like a real desktop. The applications can be moved across the desktop, like in a real one. Additionally, the files need to be double clicked in order to open. Another interesting feature is whenever clicking on a button, an iconic sound plays, just like in a real windows 98.
